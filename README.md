# Academic Website

This is a small static website that can be hosted for free on GitHub Pages.

## Files

- `index.html` contains the page content.
- `movies.html` contains the quiet personal movies subpage.
- `styles.css` contains the visual design.
- `cv.pdf` is the current CV linked from the site.

## GitHub Pages Setup

1. Create a GitHub repository named `mbeiran.github.io`.
2. Upload these files to the repository root.
3. In GitHub, open Settings -> Pages.
4. Set the source to deploy from the `main` branch.
5. The site will be available at `https://mbeiran.github.io`.

For a custom domain later, add a `CNAME` file containing the domain name and configure DNS with the domain registrar.
